## Deployment

1. Download the repository
```bash
  $ git clone https://github.com/KhinPhone060/Simpy-Note.git
  $ cd Simpy-Note
```
2. In Podfile
```bash
pod 'Floaty', '~> 4.2.0'
pod 'Toast', '~> 4.0.0'
```
3. Pod
```bash
pod install
```
4. Open the project in XCode
```bash
$ open SimpyNote.xcodeprj
```
5. Compile and run in your XCode simulator
